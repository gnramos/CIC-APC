Exercício: Produto
==================


Problema
--------

Implemente o código para ler 3 números inteiros e exibir o produto deles.


Entrada
-------

3 números inteiros.


Saída
-----

O valor resultante da multiplicação entre os números.


Exemplo(s)
----------

| Entrada | Saída |
|---------|-------|
| 1       | 6     |
| 2       |       |
| 3       |       |

| Entrada | Saída |
|---------|-------|
| 1       | -6    |
| 2       |       |
| -3      |       |

| Entrada | Saída |
|---------|-------|
| 2       | 0     |
| 1       |       |
| 0       |       |
